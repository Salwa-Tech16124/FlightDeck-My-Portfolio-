## 📡 MISSION OVERVIEW

**FlightDeck** is a highly interactive, aviation and radar-themed personal portfolio. Built with React and Framer Motion, it ditches the traditional "resume" format in favor of a **Pilot Dossier** and **System Diagnostics Dashboard**. It is designed to give recruiters and technical hiring managers an immersive, premium UX experience while reviewing technical credentials.

Featuring deep, space-black backgrounds (`#0b0e14`), glowing HUD Cyan text (`#00e5ff`), and Radar Green accents (`#00ff41`), the interface mimics the feeling of operating a high-tech Air Traffic Control console.

## 🎛 FLIGHT MODULES (SYSTEM FEATURES)

- 🛫 **Ground Control (Hero):** Dynamic typing terminal with a live horizontal "Tower Announcements" ticker and resume access.
- 🪪 **Pilot Dossier:** A 3-column glassmorphism profile card featuring verified credentials, profile imagery, and mission objectives.
- ⏱ **System Diagnostics:** Animated counter dashboard displaying real-time recruiter metrics (Projects, Focus Areas, Status).
- 🧭 **Flight Hangar:** Project showcases designed as interactive boarding passes with seamless GitHub routing.
- 📋 **Safety Protocol:** A QA philosophy statement paired with an animated verification checklist for hard skills.
- 📡 **Comms Tower:** A fully functional, Air Traffic Control-styled transmission form to establish direct communication.

## 🚀 TAKEOFF SEQUENCE (LOCAL SETUP)

To launch the FlightDeck environment locally, run the following terminal commands to gain clearance:

```bash
# 1. Clone the repository into your local airspace
git clone https://github.com/Salwa-Tech16124/FlightDeck-Portfolio.git

# 2. Enter the control center
cd FlightDeck-Portfolio

# 3. Install required flight dependencies
npm install

# 4. Ignite engines (Start Dev Server)
npm run dev
```

*The application will transmit locally on `http://localhost:5173`.*
